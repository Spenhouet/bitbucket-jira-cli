# bj issue status

Show open Jira issues assigned to you.

## Synopsis

```
bj issue status [OPTIONS]
```

## Options

| Option | Description |
| --- | --- |
| `-L, --limit <integer>` | Max results. _(default: 30)_ |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
