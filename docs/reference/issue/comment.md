# bj issue comment

Add a comment to a Jira issue.

## Synopsis

```
bj issue comment [OPTIONS] [KEY]
```

## Arguments

| Argument | Description |
| --- | --- |
| `KEY` | Issue key (default: from branch). |

## Options

| Option | Description |
| --- | --- |
| `-b, --body <text>` | Comment text. |
| `-e, --editor` | Write the body in $EDITOR. |
