# bj issue edit

Edit a Jira issue's fields.

## Synopsis

```
bj issue edit [OPTIONS] [KEY]
```

## Arguments

| Argument | Description |
| --- | --- |
| `KEY` | Issue key (default: from branch). |

## Options

| Option | Description |
| --- | --- |
| `-s, --summary <text>` | New summary. |
| `-b, --body <text>` | New description. |
| `-e, --editor` | Write the body in $EDITOR. |
