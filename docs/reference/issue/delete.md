# bj issue delete

Delete a Jira issue (irreversible).

## Synopsis

```
bj issue delete [OPTIONS] [KEY]
```

## Arguments

| Argument | Description |
| --- | --- |
| `KEY` | Issue key (default: from branch). |

## Options

| Option | Description |
| --- | --- |
| `-y, --yes` | Skip the confirmation prompt. |
