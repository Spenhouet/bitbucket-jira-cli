# bj issue fields

List the editable fields on an issue (name, id, type, value, allowed values).

## Synopsis

```
bj issue fields [OPTIONS] [KEY]
```

## Arguments

| Argument | Description |
| --- | --- |
| `KEY` | Issue key (default: from branch). |

## Options

| Option | Description |
| --- | --- |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
