# bj ruleset list

List a repository's branch restrictions.

## Synopsis

```
bj ruleset list [OPTIONS]
```

## Options

| Option | Description |
| --- | --- |
| `-R, --repo <text>` | WORKSPACE/REPO. |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
