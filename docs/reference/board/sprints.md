# bj board sprints

List a board's sprints.

## Synopsis

```
bj board sprints [OPTIONS] BOARD_ID
```

## Arguments

| Argument | Description |
| --- | --- |
| `BOARD_ID` | Board id (from `board list`). _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
