# bj board list

List Jira boards.

## Synopsis

```
bj board list [OPTIONS]
```

## Options

| Option | Description |
| --- | --- |
| `-p, --project <text>` | Filter by project key. |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
