# bj search

Search Bitbucket and Jira.

## Synopsis

```
bj search <command> [OPTIONS]
```

## Description

Search Bitbucket repositories and code and Jira issues (JQL). Jira issue search is also available as `bj issue list --jql`.

## Commands

| Command | Description |
| --- | --- |
| [`repos`](repos.md) | Search repositories in a workspace by name. |
| [`code`](code.md) | Search code across a workspace. |
| [`issues`](issues.md) | Search Jira issues with JQL. |
