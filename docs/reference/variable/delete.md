# bj variable delete

Delete a pipeline variable by name.

## Synopsis

```
bj variable delete [OPTIONS] KEY
```

## Arguments

| Argument | Description |
| --- | --- |
| `KEY` | Variable name. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-R, --repo <text>` | WORKSPACE/REPO. |
