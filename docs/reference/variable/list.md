# bj variable list

List a repository's pipeline variables.

## Synopsis

```
bj variable list [OPTIONS]
```

## Options

| Option | Description |
| --- | --- |
| `-R, --repo <text>` | WORKSPACE/REPO. |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
