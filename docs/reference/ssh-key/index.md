# bj ssh-key

Manage your Bitbucket account SSH keys.

## Synopsis

```
bj ssh-key <command> [OPTIONS]
```

## Description

Manage your Bitbucket account SSH keys. Needs a token with account scope; the default login scopes do not include it.

## Commands

| Command | Description |
| --- | --- |
| [`list`](list.md) | List your account SSH keys. |
| [`add`](add.md) | Add an SSH key to your account. |
| [`delete`](delete.md) | Delete an SSH key from your account. |
