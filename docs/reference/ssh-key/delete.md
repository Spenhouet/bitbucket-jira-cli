# bj ssh-key delete

Delete an SSH key from your account.

## Synopsis

```
bj ssh-key delete [OPTIONS] KEY_UUID
```

## Arguments

| Argument | Description |
| --- | --- |
| `KEY_UUID` | Key uuid (from `ssh-key list`). _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-y, --yes` | Skip the confirmation prompt. |
