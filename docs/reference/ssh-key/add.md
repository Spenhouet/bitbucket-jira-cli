# bj ssh-key add

Add an SSH key to your account.

## Synopsis

```
bj ssh-key add [OPTIONS] KEY_FILE
```

## Arguments

| Argument | Description |
| --- | --- |
| `KEY_FILE` | Path to the public key file. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-t, --title <text>` | Label for the key. _(required)_ |
