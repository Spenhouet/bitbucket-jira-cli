# bj pipeline list

List recent pipeline runs.

## Synopsis

```
bj pipeline list [OPTIONS]
```

## Options

| Option | Description |
| --- | --- |
| `-L, --limit <integer>` | Max results. _(default: 30)_ |
| `-R, --repo <text>` | Target repo as WORKSPACE/REPO. |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
