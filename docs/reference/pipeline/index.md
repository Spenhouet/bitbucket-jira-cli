# bj pipeline

Work with Bitbucket Pipelines.

## Synopsis

```
bj pipeline <command> [OPTIONS]
```

## Commands

| Command | Description |
| --- | --- |
| [`list`](list.md) | List recent pipeline runs. |
| [`view`](view.md) | View a pipeline run and its steps. |
| [`logs`](logs.md) | Print the logs for a pipeline's steps. |
| [`run`](run.md) | Trigger a pipeline run. |
| [`stop`](stop.md) | Stop a running pipeline. |
| [`watch`](watch.md) | Follow a running pipeline until it finishes. |
| [`rerun`](rerun.md) | Re-run a pipeline with the same target (Bitbucket has no native rerun). |
