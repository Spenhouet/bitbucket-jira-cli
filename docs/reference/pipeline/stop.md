# bj pipeline stop

Stop a running pipeline.

## Synopsis

```
bj pipeline stop [OPTIONS] PIPELINE
```

## Arguments

| Argument | Description |
| --- | --- |
| `PIPELINE` | Build number or pipeline UUID. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-R, --repo <text>` | Target repo as WORKSPACE/REPO. |
