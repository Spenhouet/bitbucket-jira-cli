# bj auth git-credential

Git credential helper: emit the Bitbucket token for bitbucket.org (get only).

## Synopsis

```
bj auth git-credential [OPTIONS] OPERATION
```

## Arguments

| Argument | Description |
| --- | --- |
| `OPERATION` | git credential operation (get/store/erase). _(required)_ |
