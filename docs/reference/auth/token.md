# bj auth token

Print the stored token for a backend (for scripting).

## Synopsis

```
bj auth token [OPTIONS] BACKEND
```

## Arguments

| Argument | Description |
| --- | --- |
| `BACKEND` | Which backend's token to print. _(required)_ |
