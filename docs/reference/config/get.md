# bj config get

Print a single configuration value.

## Synopsis

```
bj config get [OPTIONS] KEY
```

## Arguments

| Argument | Description |
| --- | --- |
| `KEY` | Dotted key, e.g. bitbucket.workspace. _(required)_ |
