# bj pr status

Show open PRs relevant to you (authored or reviewing).

## Synopsis

```
bj pr status [OPTIONS]
```

## Options

| Option | Description |
| --- | --- |
| `-R, --repo <text>` | Target repo as WORKSPACE/REPO. |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
