# bj pr checkout

Check out a pull request's source branch locally.

## Synopsis

```
bj pr checkout [OPTIONS] PR_ID
```

## Arguments

| Argument | Description |
| --- | --- |
| `PR_ID` | PR id to check out. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-R, --repo <text>` | Target repo as WORKSPACE/REPO. |
