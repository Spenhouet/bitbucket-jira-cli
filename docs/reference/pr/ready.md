# bj pr ready

Mark a draft pull request as ready for review (or --undo back to draft).

## Synopsis

```
bj pr ready [OPTIONS] [PR_ID]
```

## Arguments

| Argument | Description |
| --- | --- |
| `PR_ID` | PR id (default: current branch). |

## Options

| Option | Description |
| --- | --- |
| `--undo` | Convert back to a draft instead. |
| `-R, --repo <text>` | Target repo as WORKSPACE/REPO. |
