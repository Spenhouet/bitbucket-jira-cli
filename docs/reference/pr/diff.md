# bj pr diff

View the changes in a pull request.

## Synopsis

```
bj pr diff [OPTIONS] [PR_ID]
```

## Arguments

| Argument | Description |
| --- | --- |
| `PR_ID` | PR id (default: current branch). |

## Options

| Option | Description |
| --- | --- |
| `--stat` | Show a diffstat summary. |
| `-R, --repo <text>` | Target repo as WORKSPACE/REPO. |
