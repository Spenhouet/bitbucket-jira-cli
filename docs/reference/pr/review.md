# bj pr review

Approve, request changes on, or unapprove a pull request.

## Synopsis

```
bj pr review [OPTIONS] [PR_ID]
```

## Arguments

| Argument | Description |
| --- | --- |
| `PR_ID` | PR id (default: current branch). |

## Options

| Option | Description |
| --- | --- |
| `--approve` | Approve the PR. |
| `--request-changes` | Request changes. |
| `--unapprove` | Remove your approval. |
| `-R, --repo <text>` | Target repo as WORKSPACE/REPO. |
