# bj release delete

Delete a version.

## Synopsis

```
bj release delete [OPTIONS] VERSION_ID
```

## Arguments

| Argument | Description |
| --- | --- |
| `VERSION_ID` | Version id. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-y, --yes` | Skip the confirmation prompt. |
