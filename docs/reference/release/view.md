# bj release view

View a single version.

## Synopsis

```
bj release view [OPTIONS] VERSION_ID
```

## Arguments

| Argument | Description |
| --- | --- |
| `VERSION_ID` | Version id. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
