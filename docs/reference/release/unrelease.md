# bj release unrelease

Mark a version as unreleased.

## Synopsis

```
bj release unrelease [OPTIONS] VERSION_ID
```

## Arguments

| Argument | Description |
| --- | --- |
| `VERSION_ID` | Version id. _(required)_ |
