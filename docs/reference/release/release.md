# bj release release

Mark a version as released.

## Synopsis

```
bj release release [OPTIONS] VERSION_ID
```

## Arguments

| Argument | Description |
| --- | --- |
| `VERSION_ID` | Version id. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `--release-date <text>` | YYYY-MM-DD. |
