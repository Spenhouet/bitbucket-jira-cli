# bj release

Manage Jira project versions (releases).

## Synopsis

```
bj release <command> [OPTIONS]
```

## Description

Jira project versions, the analog of GitHub releases. A version is a named milestone that issues target with their fix version and that you eventually mark released.

## Commands

| Command | Description |
| --- | --- |
| [`list`](list.md) | List a project's versions. |
| [`view`](view.md) | View a single version. |
| [`create`](create.md) | Create a version on a project. |
| [`edit`](edit.md) | Edit a version's fields. |
| [`release`](release.md) | Mark a version as released. |
| [`unrelease`](unrelease.md) | Mark a version as unreleased. |
| [`delete`](delete.md) | Delete a version. |
