# bj release edit

Edit a version's fields.

## Synopsis

```
bj release edit [OPTIONS] VERSION_ID
```

## Arguments

| Argument | Description |
| --- | --- |
| `VERSION_ID` | Version id. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `--name <text>` |  |
| `-d, --description <text>` |  |
| `--release-date <text>` | YYYY-MM-DD. |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
