# bj repo clone

Clone a repository locally.

## Synopsis

```
bj repo clone [OPTIONS] REPO [DIRECTORY]
```

## Arguments

| Argument | Description |
| --- | --- |
| `REPO` | WORKSPACE/REPO to clone. _(required)_ |
| `DIRECTORY` | Target directory. |

## Examples

```bash
bj repo clone myteam/myrepo
```
