# bj repo file

Print the contents of a file in the repository.

## Synopsis

```
bj repo file [OPTIONS] PATH
```

## Arguments

| Argument | Description |
| --- | --- |
| `PATH` | File path in the repository. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `--ref <text>` | Branch, tag, or commit. _(default: HEAD)_ |
| `-R, --repo <text>` | WORKSPACE/REPO (default: git remote). |
