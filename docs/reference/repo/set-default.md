# bj repo set-default

Store the default workspace used when not inside a clone.

## Synopsis

```
bj repo set-default [OPTIONS] REPO
```

## Arguments

| Argument | Description |
| --- | --- |
| `REPO` | WORKSPACE/REPO to treat as default. _(required)_ |
