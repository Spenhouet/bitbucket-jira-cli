# bj repo delete

Delete a repository (irreversible; needs repository admin scope).

## Synopsis

```
bj repo delete [OPTIONS] REPO
```

## Arguments

| Argument | Description |
| --- | --- |
| `REPO` | WORKSPACE/REPO to delete. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-y, --yes` | Skip the confirmation prompt. |
