# bj repo view

View a repository.

## Synopsis

```
bj repo view [OPTIONS] [REPO]
```

## Arguments

| Argument | Description |
| --- | --- |
| `REPO` | WORKSPACE/REPO (default: git remote). |

## Options

| Option | Description |
| --- | --- |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
| `-w, --web` | Open in the browser. |
