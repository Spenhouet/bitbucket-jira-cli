# bj repo rename

Rename a repository.

## Synopsis

```
bj repo rename [OPTIONS] NEW_NAME
```

## Arguments

| Argument | Description |
| --- | --- |
| `NEW_NAME` | New repository name. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-R, --repo <text>` | WORKSPACE/REPO (default: git remote). |
