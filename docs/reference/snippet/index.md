# bj snippet

Work with Bitbucket snippets.

## Synopsis

```
bj snippet <command> [OPTIONS]
```

## Commands

| Command | Description |
| --- | --- |
| [`list`](list.md) | List snippets in a workspace. |
| [`view`](view.md) | View a snippet's metadata and files. |
| [`delete`](delete.md) | Delete a snippet. |
