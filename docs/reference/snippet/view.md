# bj snippet view

View a snippet's metadata and files.

## Synopsis

```
bj snippet view [OPTIONS] SNIPPET_ID
```

## Arguments

| Argument | Description |
| --- | --- |
| `SNIPPET_ID` | Snippet id. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-w, --workspace <text>` | Workspace (default: configured). |
| `--json` | Output raw JSON. |
| `-q, --jq <text>` | Filter JSON with a jq expression. |
