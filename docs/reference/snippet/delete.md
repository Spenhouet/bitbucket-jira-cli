# bj snippet delete

Delete a snippet.

## Synopsis

```
bj snippet delete [OPTIONS] SNIPPET_ID
```

## Arguments

| Argument | Description |
| --- | --- |
| `SNIPPET_ID` | Snippet id. _(required)_ |

## Options

| Option | Description |
| --- | --- |
| `-w, --workspace <text>` | Workspace (default: configured). |
| `-y, --yes` | Skip the confirmation prompt. |
