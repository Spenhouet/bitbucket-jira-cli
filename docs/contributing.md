# Contributing

Contributions are welcome. See the full guide in
[`CONTRIBUTING.md`](https://github.com/Spenhouet/bitbucket-jira-cli/blob/main/CONTRIBUTING.md).

In short:

```bash
git clone https://github.com/Spenhouet/bitbucket-jira-cli.git
cd bitbucket-jira-cli
uv sync --all-groups
uv run bj --help
uv run ruff check
uv run pytest
```
